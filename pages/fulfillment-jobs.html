How to Setup Recurring Jobs

A

Account-Level (Global)

All stable channels at once — Calibrate uses this for Logiwa.

Account→System Settings→Recurring Jobs
  1. 1 Open Recurring Jobs tab
  2. 2 Check Enabled
  3. 3 Set Period + Delay
  4. 4 Press Save

✅ Inventory · products · export queue

C

Webhook + Emergency Fallback

Instant webhook with scheduled backup if an event is missed.

Channels→[Channel]→Mode: Webhook
  1. 1 Set Mode to Webhook
  2. 2 Delay = None (instant)
  3. 3 Emergency Job Period = Each 15 min
  4. 4 Press Save

✅ Recommended for Logiwa orders & receipts

🎯 Key takeaways

⚡Webhook = instant sync (recommended)
↻Emergency period = 15 min fallback
🕐Account jobs = all stable channels
⚠️Receipt retry won't fix line mismatch
✗PO export = no auto-retry · manual resend
